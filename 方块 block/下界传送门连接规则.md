**常见范围**

根据代码，对于坐标为x(实数)的实体，它能搜索到的门有：  
	下界传送门：floor(x/8)-16 <= n <= floor(x/8)+16  
	主世界传送门：floor(8x)-128 <= n <= floor(8x)+128  

可以推出，传送门坐标为n(整数)，能被搜索到的实体有：  
	下界实体：(n-128)/8 <= x < (n+129)/8  
	主世界实体：8(n-16) <= x < 8(n+17)  

注意：  
1.	请注意看仔细取值范围的左右端点是开区间还是闭区间！  
2.	这几个公式仅用于计算搜索到的传送门的范围，或实体能连  接到这个门的实体位置范围，仅用于设计刷怪塔的时候确定传送门的范围  
3.	实体最终传送后的位置并不是搜索到的传送门的位置，还有考虑传送前后实体在传送门中的相对位置不变和躲避碰撞箱产生的偏移等等。 

**原理**

$$n - 128 \le \text{floor}(8x) \le n + 128$$
$$\min(K) \le 8x < \max(K) + 1, K = \text{floor}(8x)$$
$$(n - 128) \le 8x < (n + 128) + 1$$
$$n - 128 \le 8x < n + 129$$
$$\frac{n - 128}{8} \le x < \frac{n + 129}{8}$$

---
$$n - 16 \le \text{floor}\left(\frac{x}{8}\right) \le n + 16$$
$$\min(K) \le \frac{x}{8} < \max(K) + 1, K = \text{floor}(x/8)$$
$$(n - 16) \le \frac{x}{8} < (n + 16) + 1$$
$$n - 16 \le \frac{x}{8} < n + 17$$
$$8(n - 16) \le x < 8(n + 17)$$
$$8n - 128 \le x < 8n + 136$$
