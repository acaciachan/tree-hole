已知实体位置实数x，求能搜索到的门的方块坐标n：

1. 对于主世界的实体坐标为实数x的实体，
它能搜索到并前往下界的传送门的方块位置整数n取值范围为：floor(x/8)-16 <= n <= floor(x/8)+16

2. 对于下界的实体坐标为实数x的实体，
它能搜索到并前往主世界的传送门的方块位置整数n取值范围为：floor(8x)-128 <= n <= floor(8x)+128

已知传送门方块坐标整数n，求实体位置x：
1. 对于一个在主世界的方块坐标为整数n的传送门，
下界的实体能传送过来的实体坐标x满足：(n-128)/8 <= x < (n+129)/8
2. 对于一个在下界的方块坐标为整数n的传送门，
主世界的实体能传送过来的实体坐标x满足：8n-128 <= x < 8n+136

注意：
1.	请注意看仔细取值范围的左右端点是开区间还是闭区间！
2.	这几个公式仅用于计算搜索到的传送门的范围，或实体能连接到这个门的实体位置范围，仅用于设计刷怪塔的时候确定传送门的范围
3.	实体最终传送后的位置并不是搜索到的传送门的位置，还有考虑传送前后实体在传送门中的相对位置不变和躲避碰撞箱产生的偏移等等。
