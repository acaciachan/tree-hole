总掉落经验 = 基础经验 + 装备经验

基础经验：
	源代码：
    protected void dropExperience(ServerLevel level, @Nullable Entity entity) {
        if (!this.wasExperienceConsumed()
            && (
                this.isAlwaysExperienceDropper()
                    || this.lastHurtByPlayerTime > 0 && this.shouldDropExperience() && level.getGameRules().getBoolean(GameRules.RULE_DOMOBLOOT)
            )) {
            ExperienceOrb.award(level, this.position(), this.getExperienceReward(level, entity));
        }
    }
    
	解释：
		!this.wasExperienceConsumed() 用途是确认经验没有被幽匿催发体吃掉
		this.isAlwaysExperienceDropper() 玩家为true，非玩家为false，表示玩家一定掉落经验
		this.lastHurtByPlayerTime > 0 表示被玩家伤害过
		shouldDropExperience()：幼年不掉落经验(返回false)，非幼年掉落(返回true)，例外有：
			Monster(怪物)：返回true
			Hoglin(疣猪兽)：返回true
			Tadpole(蝌蚪)：返回false
	
	分析可知，掉落经验的前置条件：
	1.	如果附近有幽匿催发体把经验吃了，那后续就不掉落经验了；
	2.	没有幽匿催发体吃经验，则能掉落经验
		2.1	玩家可以掉落经验；
		2.2	非玩家只有被玩家攻击过才掉落经验，且受到“gamerule doMobLoot”控制；
				非玩家实体默认成年体掉落经验，幼年生物不掉落经验。
				但是怪物(monster)和疣猪兽的成年和幼年都会掉落，而蝌蚪成年和幼年都不掉落；		
	
	如果以上都通过，则计算应该掉落多少经验：getBaseExperienceReward()，默认为0，但实际上几乎所有的生物都是走的以下代码覆写：
		非玩家生物：总经验 = 基础经验 + 装备经验
			基础经验：
				BabyZombie(幼年僵尸类)：基础经验 = floor(2.5*基础经验) 
				Animal(动物) = 1~3
					ChickenJockey(鸡骑士的鸡) = 10
					Hoglin(疣猪兽) = 5
				Piglin(猪灵) = 5
				WaterAnimal(水生动物) = 1~3
			装备经验：
				这里的装备包含：
					手持物品(handItem)：左右手
					普通装备(armor)：从头到脚
					身体盔甲(bodyArmor)：(狼铠、马凯，目前不会自然生成)
				每一件装备都进行判断：
					如果这件装备的掉落概率<=1，每件盔甲就能转化成1~3的经验
		玩家：掉落7倍经验等级的经验值，最多掉落100经验。
