继续概率 d = 1 → p → p^2 → p^3 → p^4 → ... → p^n
达到概率 P =     1 → p   → p^2 → p^3 → ... → p^(n-1)

P(高度=n) = p^(0+1+2+...+(n-1))*(1-p^n) = p^(n(n-1)/2)*(1-p^n)
平均高度：
Sum[p^(n*(n - 1)/2), {n, 1, Infinity}]
= EllipticTheta[2, 0, Sqrt[p]]/(2 p^(1/8))
