主难度：和平(peaceful)，简单(easy)，普通(normal)，困难(haCD)

MC的最小时间单位为tick，现实世界1秒钟=20tick
MC里一天(MC日)为24000tick=20分钟，也就是白天10分钟，晚上10分钟；

两种计时器：
	世界总时间：用来表示世界已经走过了多少个tick，世界总时间 = 世界天数*24000+当日时间(/time query daytime)，
	
	区块占据时间：玩家加载的区块每tick都会使累计时间+1，多个玩家会叠加。
		玩家加载的区块(net.minecraft.server.level.ServerChunkCache#tickChunks)

区块难度三个重要参数：
	1.世界时间进度值(DT = 0 ~ 0.25)：
		世界总时间为 0~3 MC日，世界时间进度值 = 0
		世界总时间为 3~63 MC日，世界时间进度值 = 0.25*(世界总时间 - 3*24000)/(60*24000) = 0~0.25
		世界总时间为 63~+∞ MC日，世界时间进度值 = 0.25

	2.区块占据时间进度值(CIT = 0 ~ 1):
		区块占据时间小于150 MC日，CIT = 区块占据时间 / (150*24000)；
		区块占据时间大于150 MC日，CIT = 1；

	3.月光率(MoonBrightness = Moon = 0~1)：
		周期变化，从第一天开始，亮度在下面八个数字反复循环： {1, 0.75, 0.5, 0.25, 0, 0.25, 0.5, 0.75}
		详细请查看“时间”相关的文档里面的“月相和月光率”部分

区块难度计算中间过程：
		和平模式直接为0，不进行后续计算
		非和平模式：
			区块难度参数(k)：
				k = 区块占据时间进度值(CIT) + min{世界时间进度值(DT), 月光率*0.25}；
				(注意，min{}内的两个参数范围均为 0~0.25)
			
			区块难度(CD)：
				CD = 0.75 + 世界时间进度值(DT)(非困难模式再*0.75) + 区块难度参数(k)(简单模式下k再乘0.5)；
				简单模式下r再乘1，普通模式下r再乘2，困难模式下r再乘3
	
	整理后，每个难度：
	和平模式：CD = 0
	简单模式：CD = 1*(0.75 + DT + 0.5*(CIT*0.75 + min{DT,Moon*0.25}) = 0.75 ~ 1.5
	普通模式：CD = 2*(0.75 + DT + CIT*0.75 + min{DT,Moon*0.25}) = 1.5 ~ 4
	困难模式：CD = 3*(0.75 + DT + CIT + min{DT,Moon*0.25}) = 2.25 ~ 6.75
	
	实际上我们需要考虑的往往是游戏时间超过63 MC日后，世界时间进度值会饱和(DT=0.25)，有：
	和平模式：CD = 0
	简单模式：CD = 1 + 0.375*CIT + 0.125*Moon = 1 ~ 1.5
	普通模式：CD = 2 + 1.5*CIT + 0.5*Moon = 2 ~ 4
	困难模式：CD = 3 + 3*CIT + 0.75*Moon = 3 ~ 6.75
	
区块难度指数(CDM)：
	区块难度小于2，区块难度因子 CDM = 0
	区块难度为2~4，区块难度因子 CDM = (CD-2)/2
	区块难度大于4，区块难度因子 CDM = 1

	要让某个区块难度指数到达上限(CDM=1)，且不会受到月相的波动，
	首先最容易的就是要让总游戏时间达到63MC日，
	然后区块的占用时间还要：
		和平难度：CDM恒为 0
		简单难度：CIT溢出CDM依然受月相影响，CD = 1.375 + 0.125*Moon = 1.375 ~ 1.5，CDM恒为零
		普通难度：CIT溢出CDM依然受月相影响，CD = 3.5 + 0.5*Moon = 3.5 ~ 4，CDM = 0.75 ~ 1
		困难模式：CIT>=1/3(50MC天，16.67小时)，CD = 4 + 0.75*Moon = 4 ~ 4.75，CDM恒为1



区块难度(CD)影响：
	雷暴天气生成骷髅马的概率：CD%
	着火僵尸攻击玩家，引火到玩家身上的概率：CD% * 30
		着火时长：7*CD 秒
	当CD>2时，幻术师可导致玩家失明

	
区块难度指数(CDM)影响：
	僵尸成为牢大的概率为 0.05*CDM
		牢大召唤僵尸增援的数量可能更多
		牢大的初始血量和普通僵尸一样依然为20
		但是牢大的血量上限为[20,100)范围的均匀随机数
	生物可捡起掉落物的概率为 0.55*CDM
	生物装备：
		生物自带装备的概率：不通用，每种怪物不一样，详情请见“生物装备”
		生物掉落自带装备的概率为 (8.5+抢夺等级)/100
		生物自带盔甲的概率为 0.15*CDM，详细每件盔甲和材质的概率请看“生物盔甲”
		附魔相关：
			生物自带的武器附魔概率为 0.25*CDM
			生物自带的(每件)盔甲附魔概率为 0.5*CDM
			生物自带装备如果附魔，附魔等级为 5+18*CDM = 5~23
	史莱姆尺寸分布：
		小型(size=1)：(1-0.5CDM)/3 = 1/6 ~ 1/3
		中型(size=2)：1/3
		大型(size=4)：(1+0.5CDM)/3 = 1/3 ~ 1/2


月光率影响：
	在沼泽y=51~69的位置，亮度小于等于7的位置，
	如果刷怪位置亮度小于一个0~7的随机数(概率为 1-(亮度/7))，
	并且月光率大于一个0~1的随机数(概率为月光率)，
	那么有50%的概率生成一只史莱姆。
	
	总结：生成史莱姆的通过概率为
		当亮度大于7时，无法生成
		当亮度为1~7时， p = 月光率*(1-亮度/7)/2
		当亮度为0时，简化为 p = 月光率/2
